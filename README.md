### Quick Start
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/E-Shop.git
    ```
2. Open the solution in Visual Studio.
3. Restore NuGet packages:
    ```bash
    dotnet restore
    ```
4. Build and run the application:
    ```bash
    dotnet run
    ```
